Callicore\Photos
======

 Callicore is a set of PHP-GTK2 based applications all based on a common application framework.  The
 Photos application is a currently simple little app to manage photos in a gallery.

Features
---

 1. listing of photos
 1. tagging
 1. organizing in albums
 1. simple image manipulation

Future
---

For now this is a very simple application but more is planned such as syncing, comments and other features.
